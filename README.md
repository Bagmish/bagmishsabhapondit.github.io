# Personal Website

Plain HTML + CSS. No dependencies, no build step.

## Setup

1. Create a repo named `yourusername.github.io` on GitHub.
2. **Add your photo** — save it as `photo.jpg` in the root of the repo (same folder as `index.html`). A black-and-white portrait works best; the CSS applies a grayscale filter automatically.
3. Update the placeholder links in `index.html` (Substack URL, nav links).
4. Push:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/yourusername/yourusername.github.io.git
git push -u origin main
```

5. Go to **Settings → Pages**, set source to `main` / root.
6. Live at `https://yourusername.github.io`.
